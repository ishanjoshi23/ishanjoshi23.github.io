Second Year Computer Engineering Courses

Content Covered:
  • Boolean algebra
  • Combinational and Sequential Circuits
  • Organization and Operation of Microcomputers
  • Memory Addressing Modes
  • Representation of Information
  • Instruction Sets
  • Machine and Assembly Language Programming
  • Systems Programs
  • Introduction to Digital System Design using Microcomputers.
The Altera DE1-SoC FPGA is used for lab projects in this course. Verilog/SystemVerilog are used for hardware design. The ARMv7 ISA is taught as part of assembly language programming.
Content Covered:
  • Software Design & Implementation,
  • Reasoning about Software Systems
  • Abstraction and Specification of Software
  • Software Testing & Verification
  • Abstract Data Types,
  • Object-Oriented Design
  • Type Hierarchies
  • Concurrent Software Design
  • Multithreading & Client-Server Design
Java is used for all programming assignments in this course. The course is taught using the IntelliJ Idea IDE. The course also covers the basics of the Git version control system as it is used for assignments and programming tests. Students learn how to use the JUnit testing framework for unit testing. General software design paradigms are taught in this course, including object-oriented design, design patterns, and concurrent software design. Working on the Mini-Projects in this course was a great way to learn how to work in a team environment.
Content Covered:
  • Partial and Directional Derivatives
  • Maxima and Minima;
  • Lagrange multipliers and Second Derivative Test
  • Multiple Integrals and Applications
This course covers the basics of multivariable calculus. Generally a useful course for engineering students.
